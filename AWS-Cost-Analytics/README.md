# 📊 AWS Cost Analytics Dashboard  
### Power BI • SQL • Cloud Analytics Project

## 📌 Project Overview
This project analyzes AWS cloud spending using a sample billing dataset.  
The goal is to help businesses understand **where their cloud budget is going**, identify **cost spikes**, and make **data‑driven decisions** to reduce unnecessary expenses.

This project demonstrates:
- Cloud cost analysis  
- SQL querying  
- Data modeling  
- Power BI dashboard creation  
- Business insights and storytelling

## 🎯 Business Problem
Cloud costs can grow quickly and unpredictably.  
Companies need visibility into:
- Which services cost the most  
- Monthly spending trends  
- Cost spikes  
- Usage patterns  
- Opportunities to optimize  

This dashboard provides a clear, visual breakdown of AWS spending.

---

## 🧠 Key Questions Answered
1. Which AWS services cost the most  
2. How costs change month‑to‑month  
3. Which regions generate the highest cost  
4. Which cost categories (Compute, Storage, Network) dominate  
5. Are there unusual spikes in spending

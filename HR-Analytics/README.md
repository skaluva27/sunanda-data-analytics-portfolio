# 👥 HR Analytics Dashboard  
### Power BI • SQL • Employee Insights • Attrition Analysis

## 📌 Project Overview
This project analyzes employee data to understand workforce trends, attrition patterns, performance distribution, and salary insights.  
The goal is to help HR teams and leadership make informed decisions about employee retention, hiring, and workforce planning.

This project demonstrates:
- HR data analysis  
- SQL querying  
- Power BI dashboard design  
- KPI creation  
- Business storytelling  
- Workforce insights

## 🎯 Business Problem
Employee turnover is expensive and disruptive.  
Organizations need visibility into:
- Why employees leave  
- Which departments have high attrition  
- Salary and performance patterns  
- Hiring and promotion trends  
- Workforce demographics  

This dashboard provides a clear, data‑driven view of HR metrics.

---

## 🧠 Key Questions Answered
1. What is the overall attrition rate  
2. Which departments have the highest turnover  
3. How do salary levels relate to attrition  
4. Are younger or older employees more likely to leave  
5. How does job satisfaction impact retention  
6. What are the hiring and promotion trends

## 🧾 Dataset Description (hr_data.csv)
The dataset includes:

- `Age`  
- `Gender`  
- `Department`  
- `JobRole`  
- `MonthlyIncome`  
- `JobSatisfaction`  
- `PerformanceRating`  
- `YearsAtCompany`  
- `Attrition` (Yes/No)  

This is a clean, realistic HR dataset for analytics and dashboarding.

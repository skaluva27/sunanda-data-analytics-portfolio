# 🏥 Healthcare Analytics Dashboard  
### Power BI • SQL • Patient Operations • Efficiency Insights

## 📌 Project Overview
This project analyzes patient appointment data to understand operational efficiency, wait times, no‑show patterns, department performance, and patient satisfaction.  
The goal is to help healthcare administrators improve patient flow, reduce delays, and enhance service quality.

This project demonstrates:
- Healthcare operations analytics  
- SQL querying  
- Power BI dashboard design  
- KPI creation  
- Business storytelling  
- Patient‑centric insights

## 🎯 Business Problem
Healthcare facilities struggle with:
- Long patient wait times  
- High appointment no‑show rates  
- Uneven department workloads  
- Low patient satisfaction  
- Inefficient scheduling  

This dashboard provides a clear, data‑driven view of patient operations.

---

## 🧠 Key Questions Answered
1. What is the overall patient no‑show rate  
2. Which departments experience the longest wait times  
3. How do appointment types affect patient flow  
4. What factors influence patient satisfaction  
5. Which days/times have the highest patient load  
6. How efficiently are providers handling appointments

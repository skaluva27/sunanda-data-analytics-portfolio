# Retail Sales Analytics Dashboard (Power BI + SQL)

This project analyzes retail sales data using SQL and Power BI to uncover insights about revenue, profit, product performance, and regional trends. The dashboard provides interactive KPIs and visualizations to support data-driven decision-making.

## 🔍 Objectives
- Clean and prepare raw sales data
- Analyze sales performance using SQL
- Build an interactive Power BI dashboard
- Identify top products, regions, and monthly trends

## 🛠 Tools Used
- SQL Server
- Power BI
- Excel / CSV
- DAX

## 📊 Key Insights
- Top-performing regions and products
- Monthly revenue trends
- Profitability analysis
- YoY growth metrics

## 📁 Project Files
- `Retail_Sales_Cleaning.sql` — Data cleaning script
- `Retail_Sales_Analysis.sql` — Analysis queries
- `sample_retail_sales.csv` — Sample dataset
- `Dashboard_Description.txt` — Dashboard layout and visuals

## 🚀 How to Use
1. Run the SQL scripts to clean and analyze the data.
2. Load the cleaned dataset into Power BI.
3. Recreate the visuals using the dashboard description.
4. Explore insights and KPIs.
